## Modifier les températures dans les JSONs afin d'avoir les différentes alertes ##



Partie HTML 

    Bonne connaissance HTML de base mais les cours de cette année m'a permis de mieux me sensibiliser 
    au caractéristique de l'HTML 5 notamment les balises.

    J'ai décidé de faire un panneau central navigable pour y afficher les différentes températures du jour, en plus de l'historique et du graph.
    Cela permet d'avoir une interface plus claire et ordonnée tout en restant sur du "One-page".

Partie CSS

    J'avais déjà pu utiliser Bootstrap au cours d'un projet de mon DUT Informatique l'année dernière,
    mais j'avais utilisé la version en CSS pure (sans sass).

        Ainsi, cette année avec ce projet j'ai décidé de m'initier au langage sass ce qui m'a permit une plus grande modularité vis à vis de Bootstrap.
    En utilisant SASS j'ai remarqué qu'il était important de bien mettre en place son IDE afin de mieux appréhender les technologies utilisées.
    Cela passe en téléchargeant des plugins (j'utilise VSCode) "Live Server" pour synchroniser son code avec l'IDE, et "Watch SASS" pour compiler automatiquement le SASS en CSS, 
    tout cela est très pratique pour avoir un retour en direct sur navigateur.

Partie JS/JSON

    J'avais déjà travailler avec Javascipt l'année dernière sans trop approfondir les requêtes Ajax dans mon apprentissage.
    Avec ce projet j'ai pu manipuler toutes ces données depuis un json en l'appelant depuis de mon code Javascript.
    J'ai décidé d'étendre l'externalisation des données à l'historique et aux paramètres.
